一、题目

给定一个 k 位整数
N = dk−110^k−1^ + ⋯ + d110^1^ +d0 (0 ≤ di ≤ 9, i = 0,⋯,k−1, dk−1 > 0)，
请编写程序统计每种不同的个位数字出现的次数。
例如：给定 N = 100311，则有 2 个 0，3 个 1，和 1 个 3。

输入格式：

每个输入包含 1 个测试用例，即一个不超过 1000 位的正整数 N。

输出格式：

对 N 中每一种不同的个位数字，以 D:M 的格式在一行中输出该位数字 D 及其在 N 中出现的次数 M。要求按 D 的升序输出。

输入样例：

100311
结尾无空行

输出样例：

0:2
1:3
3:1
结尾无空行

二、方法1

1、思路

本题其实是考察了对于字符串的使用：
（1）重点在于将数 N 的形式 从数转换为字符（输入），这样方便我们对每一位进行计数。
（2）在计数的时候，我们需要将每一位的形式从字符转换为整形，这里就采用 a[i] - ‘0’ 的方法，实际上是通过 ASCⅡ 的加减达到目的。
